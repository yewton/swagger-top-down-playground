Swagger ペットショップ (本番)

これはペットショップサーバの例です。

Swaggerについてもっと知る、あるいは irc.freenode.net の IRC チャンネル #swagger に参加しましょう。

この例では、認可フィルタの確認用に special-key という API キーを利用できます。
